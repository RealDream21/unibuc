Ex 6


malware = program cu scop malitios
virus = program malitios aflat in calculator ce se multiplica si se raspandeste la alte calculatoare dintr-o retea prin ajutorul fisierelor.
dropper = program care faciliteaza transmiterea si instalarea de malware
downloader = program malware care descarca alte programe malware
spyware = program care spioneaza utilizatorul, monitorizeaza activitatea(apasari de tastatura, camera video, etc.)
riskware = programe legitime, dar care au vulnerabilitati 
ransomware = program care tine utilizatorul ostatic, cripteaza fisierele si opreste accesul la sistem pana cand o suma este platita
adware = program care lanseaza popup-uri/ reclame nedorite/ in locuri nedorite(random pe ecran).
worm = program malware de sine statator(nu se ascunde sub un alt fisier) care se raspandeste singur intr-o retea
obfuscare = Ascunderea codului de le decompilare prin tehnici speciale(ex schimbarea numelui functiilor)
